FAQ

Частые
вопросы

Нужно ли платить за ИИ-анализ? +
Нет. ИИ работает через Groq — это бесплатный сервис. Нужно только зарегистрироваться на console.groq.com, получить API ключ и вставить его в настройки. Лимиты Groq очень щедрые для личного использования.
Где хранятся мои данные? +
Все данные хранятся в Supabase — это облачная PostgreSQL база данных. Каждый пользователь видит только свои данные благодаря Row Level Security. Данные профиля (имя, фото) хранятся локально в браузере.
Работает ли на телефоне? +
Да, интерфейс адаптивный. Но IronTrack оптимизирован прежде всего для десктопа — на большом экране удобнее работать с таблицами тренировок и питания.
Могу ли я изменить код под себя? +
Конечно. IronTrack — это чистый HTML/CSS/JS без фреймворков. Открываешь файл в редакторе, меняешь что нужно. Никакой сборки, никаких зависимостей.
Какие упражнения можно записывать? +
В системе есть 31 предустановленное упражнение охватывающие все основные группы мышц. В будущем планируется возможность добавлять свои.
